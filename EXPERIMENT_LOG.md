# 競馬AI 実験ログ

## ルール
- モデルを変更する前に、必ず同じparquetで旧モデルのOOS ROIを計測してから変更する
- 実験は `models/exp_YYYYMMDD_名前/` に保存し、本番 `models/` を上書きしない
- 比較は必ず「同じparquet・同じ学習期間・同じOOS期間」でそろえる
- 実験後に必ずこのファイルに結果を追記する

## 実験詳細

### #1 ベースライン（2026-05-17）
- **parquet**: 旧版（997,305行、日付不明）
- **モデル**: `models/final_model.pkl`（2026-05-17付）
- **OOS ROI**: -12.7%（rank=1全買い、11,559レース）
- **特記事項**: この実験のparquetはすでに上書き済みで復元不可

---

### #2 fillna + inf修正（2026-05-26）
- **parquet**: 新版（652,002行、最終日付260524）
- **変更内容**:
  1. `01_make_features.py`: 種類1前走列15列をshift列でfillna（260302以降のNaN解消）
  2. `01_make_features.py`: `近5走_クラス補正スコア` のゼロ除算 → inf バグ修正
  3. `save_conditional_logit.py`: `開催` NaN行（supplement由来）を学習から除外
  4. `save_final_model.py`: 同上
- **OOS ROI**: +10.4%（rank=1全買い、11,559レース）
- **⚠️ 問題**: parquetが旧版(997k行)→新版(652k行)に変わっており、#1との直接比較不可
- **モデル保存先**: `models/final_model.pkl`（上書き済み）

---

### #3 タイム指数リーク修正+fillna（2026-05-26）
- **parquet**: 新版（652,002行、最終日付260524）、タイム指数を2013-2020統計で正規化
- **モデル保存先**: `models/exp_20260526_fillna/`
- **変更内容**:
  1. `deviation_score_group()` に `train_mask` 引数追加（2013-2020のみで統計計算）
  2. タイム指数を2013-2020の統計で正規化 → val/OOSへのリークなし
  3. `save_conditional_logit.py` / `save_final_model.py` に `--out-dir` 引数追加（実験分離）
  4. #2と同じ fillna・inf修正・supplement除外も適用済み
- **OOS ROI**: **+10.4%**（rank=1全買い、11,559レース）
  - 2023: +8.3%  2024: +8.0%  2025: +9.2%  2026: +26.9%
  - EV>0.05: +29.9%（6,480件）
- **考察**:
  - #2との比較: タイム指数リーク修正の前後でROI変化なし（+10.4% = +10.4%）
  - リーク修正はモデルを悪化させなかった → #2のROIはリーク由来でなかった
  - 方法論的に正しいのは#3（本番モデル候補）
