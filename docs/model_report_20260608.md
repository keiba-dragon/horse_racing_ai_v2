## モデル設計

- **モデル**: 条件付きロジット (clogit) + Isotonic calibration
- **学習**: L2=0.006、Adam optimizer、val NLLで early stopping
- **期間**: train=2013-2021、val=2022、OOS=2023-2026
- **新馬除外**: 全セグメント `クラス_rank != 1.0`（デビュー戦の前走NaNを回避）
- **特徴量選択**: Greedy forward selection、選択指標=2325（2023-24と2025の加重平均ROI、単年±30%キャップ）
- **NaN指示変数**: trainでNaN率>5%の特徴量に `_isnan` 二値変数を自動追加

## 改善経緯（重要な発見）

### 穴馬選択バイアスの発見と除去（芝短）

初期の芝短モデルで `1走前_3角`（コーナー通過順位）と `1走前_脚質_num` を特徴量に含めたところ:
- β < 0: 後方通過（差し・追い込み馬）を高評価
- 結果: 平均単勝オッズ22倍の穴馬を選択
- 2026年OOS ROI: **-65%**（壊滅的）

これらを除外することで 2026=-15.15%、25+26=-4.89% に改善。

### 騎手変更バイアスの発見と除去（芝中・ダ系）

`騎手変更` 特徴量は「急な騎手変更がある馬（人気薄）を高評価」する傾向が判明。除外により芝中の2026年ROIが -55% → -23% に改善。

### 上がり3Fのばらつき（近5走_上り3F_std）の発見（芝中・芝短）

`近5走_上り3F_std`（上がり3Fタイムの安定性）は芝コースで非常に有効:
- β = -0.20: **安定した末脚を持つ馬を高評価**
- 芝中では2026年に +41.81% という顕著な効果
- 芝短でも25+26改善に貢献

## 運用上の注意

1. **予測対象**: 指標1位の馬を全レースで購入する単勝全買い
2. **適用範囲**: 新馬戦（デビュー戦）は除外して予測
3. **OOS年次変動**: セグメントによって年次変動が大きい（特に2026年）。単年の大幅な損益は想定内
4. **次ステップ**: EVフィルタ（期待値フィルタ）の最適化により、高ROIレースのみ選択する運用を検討
