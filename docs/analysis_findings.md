# 競馬AI 分析まとめ (2026-05-17時点)

## モデル概要

- **モデル**: Conditional Logit（芝・ダート分割）+ Isotonic Calibration
- **スコア**: `score = calib - factor × (1/odds)`
  - factor = 0.00（新馬）/ 0.16（それ以外）
- **EV**: `EV = calib - (1/odds) × 0.80`（控除率20%考慮）
- **OOS評価期間**: 2023年以降（11,559レース）
- **Val期間**: 2021〜2022年

## 複勝についての結論

**複勝はどの条件でもプラスにならない。**

- rank=1全買い: -16.4%（単勝 -12.7% より悪い）
- EV > 0.10 適用: -18.5%（改善しない）
- gap ≥ 0.20: -11.0%（改善するが届かない）

**理由**: clogit rank=1 馬は市場でも人気になりやすく、複勝オッズが低く抑えられる。
高い的中率（74%）があっても平均払い戻しが低すぎてROIがマイナスになる。

> ※ `複勝配当` カラムはレース後の実際の払い戻し額なので、
> 「複勝オッズ帯フィルタ」はデータリーク（事後情報）。実運用不可。
