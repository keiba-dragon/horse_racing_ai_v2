主要な改善経緯
  • 新馬除外フィルター追加（全セグメント）
    デビュー戦は前走情報が存在しないため前走系特徴が全てNaN。クラス_rank≠1.0 フィルターを全5セグメントに適用し、NaN率を大幅削減。
  • 穴馬選択バイアスの発見と除去（芝短）
    1走前_3角（コーナー通過順位）と1走前_脚質_numがβ<0 → 差し・追い込み馬を高評価。 平均単勝オッズ22倍の穴馬を選択した結果、2026年OOS ROIが -65% に崩壊。 除外後: 2026=-15.15%、25+26=-4.89% に改善。
  • 騎手変更バイアスの発見と除去（芝中）
    騎手変更特徴量が人気薄騎手変更馬（穴馬）を高評価。 除外により芝中の2026年ROI: -55% → -23% に改善。
  • 2325選択指標の採用（全セグメント）
    選択指標を2023-24単年→2023-25加重平均（単年±30%キャップ付き）に変更。 2025年のデータを選択に反映することで2025年ROIの崩壊を防止。 2026年は未知のOOSとして維持（選択バイアス排除）。
  • 近5走_上り3F_std の発見（芝中・芝短）
    近5走_上り3F_std（上がり3Fタイムの安定性）を強制ベースに追加した芝中探索で劇的な改善。 β=-0.195: 安定した末脚を持つ馬を高評価。 芝中 25+26: -26.52% → +7.31%（+33.83pp 改善）。 2026年: -55% → +41.81%。
