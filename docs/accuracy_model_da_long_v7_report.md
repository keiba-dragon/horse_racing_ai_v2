## 4. 最大の発見：「近3走複勝率 vs 近3走勝率の逆転」

```
近3走_複勝率: β = +0.271  ← プラス（3着以内が多い = 好走継続）
近3走_勝率:   β = -0.176  ← マイナス（直近で勝ちがある = むしろ低評価）
```

**解釈**：ダート長距離では「直近3走で勝ちを持っている馬」が次走で勝ちにくい傾向がある。  
考えられる理由：
- 勝った後は斤量増・クラス昇級などで条件が厳しくなる
- 「3着以内は続くが1着は続きにくい」という過走パターン
- 人気集中による価値の相殺（ただしclogitはオッズ独立なので市場とは無関係）

→ **「堅実に上位に入り続ける馬」の方が「直近に勝った馬」より信頼できる**

## 5. 「相手レベル_平均着順」の効果

36特徴目として追加されて **+0.22%** の最後の一押し（29.80% → 30.02%）。

β=+0.134 と比較的大きい。クラス差や調整着順と異なり「そのレースにいた相手の全体水準」を反映する特徴量。強豪相手でも安定して好着順を残してきた馬を評価する。

---

## 6. タイム指数の二重性

```
近5走_タイム指数_max:  β = +0.375
近5走_タイム指数平均: β = +0.367
タイム指数_加速度:    β = -0.035（ほぼゼロ）
```

「ピーク」と「平均」の両方がほぼ同じ重みで採用されており、タイム指数の「高さ」が最重要シグナル。一方で「最近上昇中か」（加速度）はほぼ無効。  
→ ダート長距離は「今どの状態か」より「本来の実力水準」が重要
